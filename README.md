# Course Listing Platform:

A MERN full stack application where users can create, manage, and browse Courses.

# Tech Stack:

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose

# Features:

- User registration and login
- JWT-based authentication
- Full CRUD for courses
- Only course creators can edit/delete their courses
- Course detail page
- User dashboard
